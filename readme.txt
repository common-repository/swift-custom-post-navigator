> Swift Custom Post Navigator is a WordPress plugin which creates widget & displays post navigation using some CSS transforms and transitions.

Features:

* Easy to install.
* Easy to understand coding standard.
* Allow the selection of post name.
* Allow to change the no. of post to be displayed in navigation.
* Easy to enable/disable post title. 

Plugin Developed by August Infotech [Visit website](http://www.augustinfotech.com)

== Installation ==

1. Upload the 'swift-custom-post-navigator' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin from the 'Plugins' menu of your WordPress admin panel.
3. Drag and drop the "Custom Post Selection" widget to content sidebar in order to display Post Navigator.
4. Enter Post Title, Select post name, enter no. of post to be display in navigation and other parameters for your post display and save the details.
5. call sidebar function.
